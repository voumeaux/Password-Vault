PVault - Java Password Manager
A lightweight, desktop-based Password Vault application built with Java Swing. This application allows users to securely store, manage, and generate passwords through a simple graphical interface.

Features
Secure Login: Access to the vault is protected by a numeric PIN.

Password Management: Save, view, and delete account credentials (Label, Username, and Password).

Built-in Generator: Generate random 12-character alphanumeric passwords instantly.

Dynamic UI: Real-time updates to the vault display whenever entries are added or removed.

Scrollable View: A dedicated display area to manage long lists of stored credentials.
